## 🔍 Project Overview

This project is an end-to-end data analysis and visualization workflow using synthetic M&A data. It covers data generation, cleaning, transformation, and dashboard creation in Tableau.

---

## ⚙️ Workflow

### 1. Data Generation

* Synthetic dataset of 5000 M&A deals created using ChatGPT

### 2. Data Cleaning & Preparation

* Handled inconsistencies and missing values
* Created new calculated columns
* Structured dataset for analysis

### 3. Data Transformation

* Built pivot tables in Excel
* Prepared aggregated views for insights

### 4. Data Visualization (Tableau)

* Created multiple analytical sheets:

  * Sector-wise deal value
  * Deal activity trends
  * Deal type comparison
  * Deal size distribution
  * Country-wise fees

### 5. Dashboard Development

* Designed interactive dashboard with:

  * KPIs (Total Value, Total Deals, Top Sector, Large Cap %)
  * Filter actions (click to interact)
  * Clean layout and storytelling

---

## 📊 Key Insights

* Finance sector dominates deal value
* Large-cap deals account for majority of transactions (~92%)
* Deal activity remains stable with slight recent decline
* Certain countries generate significantly higher banking fees
